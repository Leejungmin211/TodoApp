## ✉️ Commit Message ✉️

| Message  | 설명                                              |
| :------: | :------------------------------------------------ |
|   feat   | 새로운 기능 추가                                  |
|   init   | 프로젝트 시작, 초기화                             |
|   fix    | 버그 수정                                         |
|   docs   | 문서 수정                                         |
|  style   | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우 |
| refactor | 코드 리팩토링                                     |
|   test   | 테스트 코드                                       |
|  chore   | 빌드 업무 수정, 패키지 매니저 수정                |
